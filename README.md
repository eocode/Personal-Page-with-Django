# Personal Page with Django

Practice Django with a page with personal info and contact

# Features
* ✔ Landing Page
* ✔ About me
* ✔ Portfolio
* ✔ Blog
* ✔ Contact me
* ✔ Admin Panel Customized
* ✔ Roles and Permissions
* ✔ Support Internationalization I18N (Es/En)
* ❎ SSR with React
* ❎ RESTFULL API
* ❎ GraphQL API
* ❎ Google Cloud Deploy

